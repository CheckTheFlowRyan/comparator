For execute sample from profiles OrdersShip_db2db.xml or OrdersShip_db2xls.xml:

1) start server - for example: SqlLocalDB.exe start "v11.0"

2) set path to DB in connection string OleDb, for example:
   Provider=SQLNCLI11.1;Server=(localdb)\v11.0;Integrated Security=SSPI;AttachDbFileName=C:\Comparator\data\Northwnd.mdf

Enjoy!
